# OlaData-Analyst-Project

📖 Overview

This project analyzes Ola ride booking data to understand booking trends, customer behavior, cancellations,
and revenue performance using Excel, SQL, and Power BI.

🎯 Objective

Analyze successful and canceled rides Identify top customers and ride patterns Evaluate vehicle-wise
performance Understand payment methods and ratings.

📊 Power BI Dashboard
👉 [View Dashboard](https://app.powerbi.com/groups/me/reports/254cd647-47bb-4e66-87bb-c2e2b9755a01/1195eb9c351be648ebed?experience=power-bi)

🛠️ Tools Used

Excel – Data cleaning
SQL – Data analysis using queries & views
Power BI – Dashboard visualization.

📊 Key Insights

Successful bookings and total revenue identified Top 5 customers based on ride frequency Average ride distance by 
vehicle type Cancellation reasons by customers and drivers UPI is a commonly used payment method Vehicle-wise customer ratings analyzed.

🔍 Analysis Performed

Created SQL views for: Successful bookings Cancellation analysis Top customers Ride distance per vehicle Ratings analysis 
Performed aggregation (SUM, AVG, COUNT) Built interactive Power BI dashboard (vehicle type, revenue, cancellation, ratings).

💡 Conclusion

The analysis provides insights into ride performance, customer behavior, and operational issues. 
These insights can help improve service quality, reduce cancellations, and increase revenue.
